# Smart Tourist Safety Monitoring System

## 🚀 Features
- Real-time GPS tracking
- Geofencing (Red, Yellow, Green zones)
- AI anomaly detection (Isolation Forest + Rules)
- SOS alert button for tourists
- Real-time admin SOS popups and alert list via Socket.IO
- Admin dashboard
- Secure login system

## 🧠 AI Logic
- Speed-based anomaly (>150 km/h)
- Distance jump detection
- Machine learning model (Isolation Forest)

## 🛠 Tech Stack
- Flask
- Flask-SocketIO
- SQLite
- JavaScript
- Bootstrap

## ▶️ Run Locally
```bash
venv\Scripts\Activate.ps1

pip install -r requirements.txt

python app.py

ctr+C

deactivate

```
